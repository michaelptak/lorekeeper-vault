> [!Danger] Obsidian Web Viewer Security
> By default links open in web viewer. This can be really convenient, but for anything you plan on logging into I would recommend right clicking > open link in default browser.
> As per the [Obsidian Docs](https://help.obsidian.md/plugins/web-viewer) third-party plugins are not sandboxed and therefore it is an additional security concern as they could *in theory* steal log-in tokens.
> Although not likely, it is definitely best practice.
